关于本站

这是一个基于 Flask 构建的个人博客系统，前后端不分离，使用服务端模板渲染。

用户系统

支持注册、登录、密码加密存储，Flask-Login 会话管理。

博客管理

支持 Markdown 编辑、分类标签、封面图上传、文章 CRUD。

评论互动

用户登录后即可对任意文章发表评论。

MySQL 5.7

使用 SQLAlchemy ORM + Flask-Migrate 管理数据库。

现代 UI

渐变配色 + 卡片布局 + 响应式设计，桌面/移动端友好。

安全防护

CSRF 防护、XSS 过滤（bleach）、密码哈希存储。

技术栈

Flask 3.0 SQLAlchemy Flask-Login Flask-WTF Flask-Migrate MySQL 5.7 Bootstrap Icons Markdown Bleach
